# `zoo-bot setup`

Initialize `~/.zoo-bot/zoo-bot.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
zoo-bot setup
zoo-bot setup --workspace ~/.zoo-bot/workspace
```

To run the wizard via setup:

```bash
zoo-bot setup --wizard
```
